# Случайная задача

Минимальное мобильное одностраничное веб-приложение для выбора случайной доступной задачи по блоку.

## Запуск

1. Откройте терминал в папке проекта.
2. Запустите простой статический сервер:

```bash
python3 -m http.server 8000
```

3. Откройте в браузере: <http://localhost:8000>

## Что умеет приложение

- Выбор блока: **Учеба / Уборка / Готовка / Уход**.
- При первом запуске список задач пустой.
- Кнопка **«Добавить задачу»** создает задачу для выбранного блока c полями:
  - название;
  - периодичность в днях;
  - `last_done: null`.
- Под формой показывается список задач выбранного блока:
  - название;
  - периодичность;
  - дата последнего выполнения или **«никогда»**;
  - кнопка **«Удалить»**.
- Если в выбранном блоке пока нет задач, показывается сообщение: **«В этом блоке нет задач»**.
- Кнопка **«Выбрать»** выбирает случайную доступную задачу по прежней логике периодичности.
- Кнопка **«Сделано»** появляется только после выбора задачи и записывает выполнение на сегодня.
- Кнопка **«Сбросить данные»** полностью очищает `localStorage`.
- Данные хранятся в `localStorage` по ключу `random_task_picker_tasks_v1`.
