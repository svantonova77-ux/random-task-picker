# Случайная задача

Минимальное мобильное одностраничное веб-приложение для выбора случайной доступной задачи по блоку.

## Запуск

1. Откройте терминал в папке проекта.
2. Запустите простой статический сервер:

```bash
python3 -m http.server 8000
```

3. Откройте в браузере: <http://localhost:8000>

## Что умеет приложение

- Выбор блока: **Учеба / Уборка / Готовка / Уход**.
- Кнопка **«Выбрать»** выбирает случайную доступную задачу.
- Кнопка **«Сделано»** появляется только после выбора задачи и записывает выполнение на сегодня.
- Кнопка **«Сбросить данные»** восстанавливает стартовый набор задач.
- Данные хранятся в `localStorage`.
